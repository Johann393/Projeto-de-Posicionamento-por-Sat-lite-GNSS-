# EE400
Este projeto explora os princípios matemáticos e computacionais por trás de sistemas de posicionamento por satélite (GNSS), com foco na modelagem de órbitas, cálculo de posição e localização tridimensional de um receptor a partir de sinais transmitidos por satélites.

A aplicação simula o funcionamento de sistemas como GPS e Galileo, utilizando parâmetros orbitais reais para descrever a trajetória dos satélites ao redor da Terra e estimar a posição de um receptor (por exemplo, um drone) a partir de medições de distância.
